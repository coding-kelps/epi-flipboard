# Security Policy

## Supported Versions
This project is a school project and is not actively maintained.
Only the latest version in the repository is available.

## Reporting a Vulnerability
If you believe you have found a security issue, please do **not** open a public issue.

Instead, report it by:
- Opening a private security advisory on GitHub, or
- Sending a mail to security@kelps.org

There is no guaranteed response time, but reasonable reports will be reviewed if possible.

## Scope
This project is provided for educational purposes only.
No security guarantees are made.
